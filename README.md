## Overview
This project demonstrates core algorithm fundamentals in Java by sorting a dataset
using Bubble Sort and then searching for a target value using Binary Search.

The program prints the low/high/mid indices during the search to show how Binary
Search reduces the search space each iteration.

---

## Project Features
- Accepts 11 integers as user input
- Sorts the dataset using Bubble Sort
- Searches for a target value using Binary Search
- Outputs intermediate search steps (low/high/mid) for transparency
